# Response Protocol

## Comment classes

- `new analysis`: needs additional empirical work
- `clarification`: the paper can likely answer this with clearer text
- `rewrite`: structure or framing needs revision
- `disagreement`: requires author judgment and careful pushback
- `minor`: typo, citation, notation, formatting

## Response structure

1. quote or summarize the comment
2. acknowledge the concern
3. state the response
4. point to the paper change

## Disagreement rules

- never write "the referee is wrong"
- prefer "we respectfully clarify" or "we have clarified why"
- escalate disagreement items for human review

## Tracking output

Use a tracker with:

- referee id
- point number
- class
- status
- manuscript location
