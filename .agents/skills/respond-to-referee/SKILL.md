# Respond To Referee

Use this skill when reviewer comments need routing, triage, and a polished response rather than a raw rebuttal.

## Workflow

1. Read the referee reports and the current paper or memo.
2. Split comments into atomic points.
3. Classify each point:
   - new analysis
   - clarification
   - rewrite
   - disagreement
   - minor edit
4. Draft a response that acknowledges the point, states the action taken, and points to the manuscript change.

## Output rules

- Flag disagreements clearly for human review.
- Do not pretend new analyses were completed if they were not.
- Keep the tone diplomatic and specific.
