# Submit

## Workflow

1. Confirm the target journal and manuscript entry point.
2. Run the pre-submission checks:
   - paper review
   - replication audit
   - citation cleanup
3. Stop if blockers remain.
4. Draft only the materials that follow from a passing gate.
