# Format Guide

## Format matrix

| Format | Slides | Duration | Core content |
|---|---:|---:|---|
| job-market | 40-50 | 45-60 min | full story, main design, mechanisms, robustness |
| seminar | 25-35 | 30-45 min | question, design, main result, selective robustness |
| short | 10-15 | 15 min | question, method, key result, implication |
| lightning | 3-5 | 5 min | hook, one result, so what |

## Deck checklist

1. first two slides explain why anyone should care
2. one key result slide anchors the talk
3. notation matches the paper
4. table density is cut for projection
5. backup slides are outside the main arc

## Review checks

- any claim on slides appears in the paper
- slide count fits the format
- figures are readable at projection scale
- no result is introduced without context
