# Data Assessment

## Compare datasets on

- concept fit
- unit of observation
- time coverage
- geography
- sample restrictions
- access conditions
- merge potential
- design compatibility

## Output table

```markdown
| Dataset | Coverage | Access | Best variables | Main limitation | Design fit |
|---|---|---|---|---|---|
```

## Decision rule

Prefer the dataset that best matches the design and claim, not the one with the largest raw size.
