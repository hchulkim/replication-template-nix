# Journal Selection

## Inputs to assess

- field and subfield
- novelty level
- identification credibility
- data uniqueness
- policy relevance
- current polish of the draft

## Output shape

```markdown
## Tier 1
- Journal: fit + main risk

## Tier 2
- Journal: fit + main risk

## Tier 3
- Journal: fit + main risk
```

## Selection rules

- do not recommend top general-interest journals by default
- separate ambition from fit
- name the likely objection for each venue
