# Target Journal

Use this skill when the question is "where should this paper go?" rather than "how do we improve the draft?"

## Workflow

1. Read the paper or abstract.
2. Identify topic, contribution type, design, and likely audience.
3. Rank journals by fit in tiers:
   - reach
   - strong field
   - safer alternatives
4. Explain why each recommended journal fits and what the main risk is.

## Output rules

- Be honest about fit and desk-rejection risk.
- Distinguish broad-interest journals from field journals.
- Prefer a ranked shortlist over a long undifferentiated list.
