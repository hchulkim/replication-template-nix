# Literature Review

Use this skill for synthesis, not citation dumping. The goal is to explain what the literature says, where papers disagree, and what gap the current project fills.

## Workflow

1. Start from local material first:
   - manuscript draft
   - notes in `quality_reports/`
   - bibliography files
   - existing review memos
2. If outside search is needed, favor primary or publisher-hosted sources and recent surveys.
3. Group papers by question or mechanism, not by publication year.
4. End with a contribution memo that states:
   - the closest papers
   - the margin of difference
   - what evidence this project adds

## Output rules

- Prefer an annotated table or short memo over long prose.
- Separate established facts from your inference about the gap.
- Quote sparingly and keep citations precise.
