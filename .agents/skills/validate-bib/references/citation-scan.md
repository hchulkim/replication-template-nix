# Citation Scan

## Files to scan

- `output/paper/*.tex`
- `output/paper/sections/*.tex`
- `output/slides/*.tex`
- `Quarto/*.qmd`

## Citation patterns

- `\cite{...}`
- `\citet{...}`
- `\citep{...}`
- `@key`
- `[@key]`

## Report sections

- missing bibliography keys
- unused bibliography entries
- possible typos or near-matches
- malformed entries if obvious
